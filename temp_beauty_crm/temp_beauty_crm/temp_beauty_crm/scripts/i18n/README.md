# Команды для работы с переводами (i18n)

## 1. Извлечение ключей из кода (создание/обновление файлов переводов)
```bash
cd frontend
npm run i18n:extract
```
**Что делает:** Сканирует весь код и создаёт/обновляет JSON файлы переводов для всех языков

---

## 2. Синхронизация ключей между языками
```bash
cd frontend
npm run i18n:sync
```
**Что делает:** Добавляет отсутствующие ключи во все языковые файлы (копирует из русского)

---

## 3. Проверка переводов на ошибки и пропущенные ключи
```bash
python3 scripts/i18n/analyze_translations.py
```
**Что делает:** 
- Проверяет наличие всех ключей во всех языках
- Находит пустые значения
- Проверяет корректность переменных `{{variable}}`
- **НЕ проверяет язык текста** (это делает следующий скрипт)

---

## 4. Проверка языка переводов (новый скрипт - нужно создать)
```bash
python3 scripts/i18n/validate_language.py
```
**Что делает:**
- Проверяет что переводы написаны на правильном языке
- Пропускает переменные в `{{}}`
- Игнорирует английские технические термины
- Выводит подозрительные строки

---

## 5. Автоматический перевод отсутствующих ключей
```bash
python3 scripts/i18n/translate_missing.py
```
**Что делает:**
- Находит пустые/отсутствующие переводы
- Переводит их с русского на целевой язык
- Использует Google Translate API (бесплатный)

---

## 6. Полный цикл (рекомендуемая последовательность)
```bash
# Шаг 1: Извлечь ключи из кода
cd frontend && npm run i18n:extract

# Шаг 2: Синхронизировать между языками
npm run i18n:sync

# Шаг 3: Перевести отсутствующие
cd .. && python3 scripts/i18n/translate_missing.py

# Шаг 4: Проверить на ошибки
python3 scripts/i18n/analyze_translations.py

# Шаг 5: Проверить правильность языка
python3 scripts/i18n/validate_language.py
```
